# 🏠 個人房貸試算器 (Personal Mortgage Calculator)

本專案是一個基於 C# Windows Forms 開發的桌面應用程式，旨在幫助使用者透過輸入房屋總價、自備款、利率等條件，精確試算出房貸的各項明細與總支出。

## ✨ 專案簡介與特色 (Features)

* **彈性的自備款輸入機制**：提供下拉式選單，支援「自備款比例 (%)」與「自備款金額 (元)」兩種模式無縫切換。
* **精確的財務計算**：支援「寬限期」設定（前幾年只繳利息不繳本金），計算結果均精確至小數點後兩位，並包含千分位逗號格式。
* **完善的防呆與驗證機制**：嚴格攔截非數字、負數、空白輸入，以及「自備款大於總價」、「寬限期大於貸款年限」等不合理邏輯，確保程式不會崩潰 (Crash)。
* **現代化 UI 設計**：
    * 利用 Panel 組合技打造「內建單位標籤 (如 %、元、年)」的專業輸入框。
    * 淺灰色浮水印提示字元 (Placeholder) 與貼心的 Tab 鍵焦點切換順序。
    * 防跑版設計：鎖定視窗大小並停用最大化功能。

## 📥 輸入與輸出項目 (I/O Specifications)

### 📌 必須輸入資訊
1. **房屋總價** (當地貨幣) 
2. **自備款** (可選比例或金額) 
3. **貸款利率** (年利率 %) 
4. **貸款年限** (年) 
5. **寬限期** (年) 

### 📊 結算顯示資訊
1.**貸款總金額** 
2.**每月應繳金額** (含寬限期前後的變化)
3.**首期利息** 與 **首期本金** 
4.**總利息支出** 
5.**總還款金額** 

## 🚀 開發環境與執行說明 (How to Run)

* **開發框架**：.NET (C# Windows Forms)
* **開發工具**：Visual Studio
* **執行步驟**：
    1.  將本專案 Clone 或下載至本機電腦。
    2.  使用 Visual Studio 開啟 `.sln` 方案檔。
    3.  按下 `F5` 或點擊上方的「開始」按鈕建置並執行應用程式。
